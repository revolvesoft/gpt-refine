Get Started with GPT Refiner

Refine your Slack messages using the power of GPT-3 with the GPT Refiner app. Follow the instructions below to get started and start using the app.

Setup Instructions

  1. Click the "Add to Slack" button above to begin the setup process.
  2. Follow the prompts to authorize the GPT Refiner app in your Slack workspace.
  3. Once authorized, the app will be installed, and you can start using the /gptrefine command in your Slack channels.

Usage Instructions

To use the GPT Refiner app, simply type the following command in any Slack channel:

/gptrefine [your message]

For example, to refine the message "I need help with my project", you would type:

/gptrefine I need help with my project

The app will respond with a refined version of your message, leveraging the capabilities of GPT-3.
